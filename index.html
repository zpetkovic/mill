Welcome to 'Who Wants to be a Millionaire' Quiz Edition

This unoffical game has 100s of questions on a range of topics. The 2 main game modes are explained below:

  • Practice mode: also known as single player, this mode is designed for you to test your skills against the computer. All lifelines are fully simulated
  • Quiz mode: this mode is designed for group quizzes. You need 2 screens to use this mode to its potential - one for the controller and one for the player. The controller must input all lifeline results.
